# 📝 Hong Kong Travel Guide | 香港旅游指南

一个全面的香港旅游信息平台，汇集了香港的主要景点、旅游路线、民䯶历史和实用旅游资䯶。

## 📱 项目特点

- 🗺 **旅游信息** - 丰富的香港旅游指南和推荐
- 📢 **景点简介** - 勺量器物分頻詳把第一次您需要知道的相关信息
- 🗍 **会论区** - 旅游经验分享和推荐
- 🌐 **在线访问** - 自托管域名访问

## 🚀 快速开始

### 在线查看
香港旅游指南: [https://xgyzbb.github.io/HKTravel/](https://xgyzbb.github.io/HKTravel/)

### 本地运行
1. Clone 仓库：
```bash
git clone https://github.com/xgyzbb/HKTravel.git
cd HKTravel
```

2. 打开 `index.html` 文件
```bash
# 方式一：直接在浏览器中打开
open index.html

# 方式二：使用本地服务器（推荐）
python -m http.server 8000
# 然后访问 http://localhost:8000
```
